Blood Grouping Detection Using Image Processing

Abstract


Determining blood type is essential including in emergency situation. Currently, these tests are performed manually by technicians, which can lead to human errors. Various systems have been developed to automate these tests, but none is able to perform the analysis in time for emergency situations. This work aims to develop an automatic system to perform the tests in a short period of time, adapting to emergency situations. In this project, A spectrum images of hand for detection taken as input and we are applying inception-v3 model on the input images and classify the blood group.
